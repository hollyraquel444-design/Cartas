Calculadora basica

El codigo de python introduce la creación de una calculadora orientada a objetos desarrollada en Python para superar las limitaciones de las calculadoras tradicionales. Esta herramienta avanzada está diseñada para realizar operaciones con tres valores en cualquier orden, procesando la entrada de izquierda a derecha. Además de las funciones aritméticas básicas, su característica distintiva es el mantenimiento de un historial detallado de las operaciones realizadas, lo que la convierte en una solución más flexible y esencial para usuarios en campos como la educación, la ingeniería y las finanzas que requieren un registro preciso de sus cálculos.
